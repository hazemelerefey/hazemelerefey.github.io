## Overview
A high-performance modern dashboard built with Next.js, TailwindCSS, and interactive Recharts.
Created specifically to provide deep insights into E-Commerce metrics, sales velocity, and inventory tracking.

## Highlights
- **Server Side Rendering (SSR)**: Utilizing Next.js for blazing fast performance.
- **Dynamic Visuals**: Recharts powers the beautiful bar and line visual components.
