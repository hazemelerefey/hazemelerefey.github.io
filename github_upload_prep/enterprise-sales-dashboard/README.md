## Overview
A React-based comprehensive admin interface connecting to real-time sales APIs. This dashboard provides deep insights into Enterprise KPIs, enabling executives to monitor performance across multiple regions instantly.

## Key Features
- **Real-Time Analytics:** WebSockets integration for live data updates.
- **Interactive Charts:** Built with Recharts for stunning visual data interpretation.
- **Responsive Design:** Fully fluid layout adapting securely to desktop and mobile views.
- **Dark Mode Support:** A sleek dark theme for data-heavy readability.
