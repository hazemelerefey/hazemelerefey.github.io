About This Project

This project focused on operational efficiency and patient flow management, moving from raw data to actionable medical insights. It spans the entire data prep lifecycle—cleaning messy clinical data, handling null values, and standardizing 'Case Types'.

The Toolkit & Roadmap

  • Power BI

    Used for core visualization and dashboard structure.

  • Power Query (M)

    Automated the ETL Pipeline (cleaning data, standardizing formats).

  • Star Schema

    Dimensional Modeling linking 'Fact' tables with 'Dimension' tables (Date, Dept, Demographics).

  • DAX

    Complex measures for Time-Band vs. Age Profiling and dynamic waitlist trends.

  • PowerPoint (UI/UX)

    Designed the custom dashboard background and split-view layout.
